Um die vorzeitige Abschaltung und Löschung deines Servers zu verhindern, empfehlen wir die Verlängerung mindestens **5 Tage vor Ablauf** durchzuführen.

## Verlängerung im Kundenportal

Um deinen Server zu verlängern, befolge diese Schritte:

1. Rufe deinen **Server** im [Kundenportal](https://app.virtbase.com/servers) auf.
2. Klicke in der Navigation auf **Plan**.

![Seite Plan aufrufen](/assets/static/articles/renew-server-step-1-de.png)

3. Wähle nun das gewünschte Paket aus. Bei einer regulären Verlängerung ist bereits das aktuelle Paket vorausgewählt.
4. Bestätige die Verlängerung mit dem Klick auf **Verlängern**. Du wirst im Anschluss zur Bezahlseite weitergeleitet.

![Paket verlängern](/assets/static/articles/renew-server-step-2-de.png)

Nach der erfolgreichen Bezahlung verlängert sich dein Server um weitere 30 Tage. Die Rechnung dazu wird im Kundenportal hinterlegt und per E-Mail an dich gesendet.

Falls du weitere Fragen zur Verlängerung hast oder Unterstützung dabei benötigst, dann [kontaktiere unser Support-Team](/de/contact).

## Was passiert wenn ich meinen Server nicht verlängere?

Wir senden dir **5 Tage** vor Ablauf der Laufzeit deines Servers eine E-Mail mit einer Erinnerung deinen Server zu verlängern.

Zum Ende der Laufzeit wird der Server mit einer weiteren Benachrichtigung **suspendiert**. Das bedeutet er kann weder gestartet werden, noch können andere
relevante Aktionen wie eine Wiederherstellung des Backups, Neuinstallation oder Änderungen an der Firewall, durchgeführt werden.

Wenn dein Server suspendiert wurde, dann hast du **5 weitere Tage**, um eine Verlängerung durchzuführen. Danach wird dein Server mit einer weiteren Ankündigung permanent gelöscht. 
Bitte beachte, dass wir in diesem Fall keine Daten wiederherstellen können.
