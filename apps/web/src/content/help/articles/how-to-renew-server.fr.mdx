Pour éviter que votre serveur ne soit arrêté et supprimé prématurément, nous recommandons de renouveler au moins **5 jours avant l'expiration**.

## Renouvellement dans le portail client

Pour renouveler votre serveur, suivez ces étapes :

1. Accédez à votre **serveur** dans le [portail client](https://app.virtbase.com/servers).
2. Cliquez sur **Plan** dans la barre de navigation.
3. Sélectionnez maintenant le forfait souhaité. Pour un renouvellement régulier, le forfait actuel est déjà présélectionné.
4. Confirmez le renouvellement en cliquant sur **Renouveler**. Vous serez ensuite redirigé vers la page de paiement.

Après un paiement réussi, votre serveur sera renouvelé pour 30 jours supplémentaires. La facture sera stockée dans le portail client et vous sera envoyée par e-mail.

Si vous avez d'autres questions sur le renouvellement ou avez besoin d'aide, veuillez [contacter notre équipe d'assistance](/fr/contact).

## Que se passe-t-il si je ne renouvelle pas mon serveur ?

Nous vous enverrons un e-mail **5 jours** avant l'expiration de votre serveur avec un rappel pour le renouveler.

À la fin du terme, le serveur sera **suspendu** avec une autre notification. Cela signifie qu'il ne peut pas être démarré, et d'autres
actions pertinentes telles que la restauration de la sauvegarde, la réinstallation ou les modifications du pare-feu ne peuvent pas être effectuées.

Si votre serveur a été suspendu, vous avez **5 jours supplémentaires** pour le renouveler. Après cela, votre serveur sera définitivement supprimé avec une autre notification.
Veuillez noter que nous ne pouvons pas restaurer de données dans ce cas.
