Het komt vaak voor dat vereisten veranderen en je een ander besturingssysteem op je server nodig hebt.
Je kunt het besturingssysteem van je server met een paar klikken en binnen enkele seconden wijzigen in het klantenportaal.

## Hoe start ik de herinstallatie?

Volg deze stappen om je server te resetten:

<Hint>
Bij een herinstallatie van het besturingssysteem worden alle gegevens onherroepelijk verwijderd.
We raden je aan eerst een back-up van de huidige configuratie te maken.
</Hint>

1. Ga naar je **server** in het [klantenportaal](https://app.virtbase.com/servers).
2. Klik in het navigatiemenu op **Overzicht**.
3. Klik aan de linkerkant op **Besturingssysteem wijzigen**.
4. Hier kun je je nieuwe besturingssysteem selecteren en een nieuw root-wachtwoord instellen. 
Je root-wachtwoord wordt niet opgeslagen en kan ook niet opnieuw worden weergegeven, dus onthoud het goed.

![Dialoogvenster Besturingssysteem resetten](/assets/static/articles/how-to-change-operating-system-nl.png)

5. Bevestig nu de reset om het proces te starten.

Je server wordt vervolgens gestopt en het besturingssysteem wordt gewijzigd.

Als je nog vragen hebt over onze besturingssystemen of hulp nodig hebt, neem dan [contact op met ons supportteam](/nl/contact).

## Hoe kan ik mijn eigen ISO-bestanden uploaden?

Momenteel is deze functie niet beschikbaar, maar we hebben een grote selectie aan kant-en-klare besturingssystemen. We zijn al bezig met de implementatie.
