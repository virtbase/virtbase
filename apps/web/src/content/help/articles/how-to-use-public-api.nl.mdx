De openbare Virtbase API biedt een flexibele manier om je geboekte diensten programmatisch te beheren. 
Hiervoor is het aanmaken van een API-sleutel nodig om geauthenticeerde verzoeken te verzenden.

## Hoe maak ik een API-sleutel aan?

Volg deze stappen om een API-sleutel aan te maken.

<Hint>
API-sleutels hebben geen vervaldatum, maar kunnen na het aanmaken slechts één keer worden weergegeven.
</Hint>


1. Ga naar de instellingen **Account** in het [klantenportaal](https://app.virtbase.com/account/settings).
2. Klik in het navigatiemenu op **API**.
3. U ziet nu een lijst met uw reeds actieve API-sleutels. Klik hier op **API-sleutel aanmaken**.
4. Geef uw sleutel een naam en bevestig de aanmaak.
5. Uw API-sleutel wordt nu eenmalig weergegeven in het bovenste gedeelte. Bewaar deze zorgvuldig.

Je API-sleutel kan direct worden gebruikt voor API-verzoeken. Als je je sleutel niet meer nodig hebt, kun je deze op elk moment verwijderen.

Als je nog vragen hebt over de openbare API of hulp nodig hebt, neem dan [contact op met ons ondersteuningsteam](/nl/contact).

## Hoe gebruik ik de openbare API?

Je vindt de volledige documentatie van alle API-eindpunten op: https://virtbase.com/api/docs.

De documentatie is alleen beschikbaar in het Engels.

Alle verzoeken aan de eindpunten moeten worden gedaan met de header `X-Virtbase-API-Key`.

## Is er een limiet?

Ja, met een API-sleutel kun je maximaal 120 verzoeken per minuut (= 2 verzoeken per seconde) naar de API sturen.

Een hogere limiet is alleen mogelijk na goedkeuring. Neem hiervoor contact op met onze support.
