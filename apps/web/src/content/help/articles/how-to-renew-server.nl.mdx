Om te voorkomen dat je server voortijdig wordt uitgeschakeld en verwijderd, raden we aan om minstens **5 dagen voor de vervaldatum** te verlengen.

## Verlenging in het klantenportaal

Volg deze stappen om je server te verlengen:

1. Ga naar je **server** in het [klantenportaal](https://app.virtbase.com/servers).
2. Klik op **Plan** in de navigatiebalk.
3. Selecteer nu het gewenste pakket. Voor een reguliere verlenging is het huidige pakket al voorgeselecteerd.
4. Bevestig de verlenging door op **Verlengen** te klikken. Je wordt vervolgens doorgestuurd naar de betaalpagina.

Na succesvolle betaling wordt je server voor nog eens 30 dagen verlengd. De factuur wordt opgeslagen in het klantenportaal en per e-mail naar je verzonden.

Als je nog vragen hebt over verlenging of hulp nodig hebt, [neem dan contact op met ons supportteam](/nl/contact).

## Wat gebeurt er als ik mijn server niet verleng?

We sturen je **5 dagen** voordat je server verloopt een e-mail met een herinnering om je server te verlengen.

Aan het einde van de termijn wordt de server **opgeschort** met een nieuwe melding. Dit betekent dat deze niet kan worden gestart, noch kunnen andere
relevante acties zoals het herstellen van de backup, herinstalleren of wijzigingen aan de firewall worden uitgevoerd.

Als je server is opgeschort, heb je nog **5 dagen** om te verlengen. Daarna wordt je server definitief verwijderd met een nieuwe melding.
Let op: we kunnen in dit geval geen gegevens herstellen.
